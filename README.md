My PT World is a responsive and interactive website created during my internship at Neuronix AI, blending my passion for Physical Therapy and Web Development. It serves as a personal platform to share study resources, insights, and motivational content connecting movement, healing, and learning.

Project Overview:
A personal platform representing my journey as a Physical Therapy student
Provides study resources, tips, and motivational thoughts
Features a fully responsive design with an interactive sidebar menu

Features:
Responsive layout for desktop and mobile screens
Toggleable sidebar menu for easier navigation
Smooth hover animations for a better user experience
Clean, reusable CSS for scalability

Tech Stack Used:
HTML
CSS
JavaScript 

How to Run Locally:

Clone the repository:
git clone https://github.com/fatimathawerani/Neuronix_AI_Project.git

Navigate to the project folder:
cd Neuronix_AI_Project

Open index.html in your browser or run a local server (e.g., Live Server extension in VS Code):
live-server

Explore the responsive layout and sidebar menu
